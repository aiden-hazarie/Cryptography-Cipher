## Cryptography Cipher
Description: A tool that encrypts and decrypts secret messages using a special code (cipher). It takes normal text and scrambles it into unreadable gibberish for privacy, and can also turn it back into the original message when needed. Think of it like a digital lockbox for sensitive information.

## Key Features
- Converts messages into secret code to prevent unauthorized reading.
- Uses a secure method (like Caesar cipher or AES encryption) to scramble data.
- Can decode messages when the right key is provided.
